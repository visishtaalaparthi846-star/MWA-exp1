HTML Lists Demonstration

This page shows the three main types of HTML lists: ordered lists, unordered lists, and definition lists.

1. Ordered List

An ordered list is used when the sequence of items is important.

Example: Steps to Submit an Assignment

  1. Open the college portal
  2. Log in with your student ID and password
  3. Select the subject
  4. Upload the assignment file
  5. Click the submit button
  6. Wait for the confirmation message

Example: Top 5 Programming Languages to Learn

  1. Python
  2. Java
  3. JavaScript
  4. C++
  5. PHP

2. Unordered List

An unordered list is used when the order of items does not matter.

Example: Grocery Shopping List

  • Rice
  • Milk
  • Bread
  • Eggs
  • Vegetables
  • Fruits

Example: Items Needed for a Computer Lab

  • Desktop Computers
  • Keyboard
  • Mouse
  • Monitor
  • Internet Connection
  • Projector

3. Definition List

A definition list is used to display terms and their meanings or descriptions.

Example: Web Development Terms

HTML
HyperText Markup Language used to create the structure of a web page.
CSS
Cascading Style Sheets used to design and style web pages.
JavaScript
A scripting language used to make web pages interactive.
Browser
A software application used to open and view websites.
URL
Uniform Resource Locator, the address of a webpage on the internet.

4. Nested List Example

Lists can also be placed inside other lists.

Example: Course Subjects and Topics

  1. Web Technology
    • HTML
    • CSS
    • JavaScript
  2. Programming
    • C Language
    • Java
    • Python
  3. Database
    • DBMS Basics
    • SQL Queries
    • Normalization

Conclusion

Ordered lists are used for steps or rankings, unordered lists are used for general item collections, and definition lists are used for terms with explanations.
